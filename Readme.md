# Github Finder App

This is my second course project from the React Front to Back 2022

## Run Locally

Clone the project

```bash
  git clone https://github.com/hassancodess/github-finder.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
